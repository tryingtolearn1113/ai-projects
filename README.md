# AI Learning Projects

## Background

I worked as an operator at Pohang Accelerator Laboratory (PAL),
While working, I thought it would be better if they operated in PyEPICS
so I began learning Python and AI to modernize them.
These projects are the result of that learning process.

## Projects

- **rag.py** — RAG system that answers questions from your own documents
- **chatbot.py** — AI chatbot that remembers conversation history  
- **summarizer.py** — reads any text file and summarizes it
- **epics_chatbot.py** — chatbot trained on live EPICS documentation,
  combining RAG retrieval with conversation memory

## Technologies

Python, LangChain, ChromaDB, Google Gemini API, BeautifulSoup

## Goal

Building toward an automatic EPICS-to-Python converter using AI.
